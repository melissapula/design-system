# Design System

Welcome to the design system. Components are built with [Lit](https://lit.dev/)
and consume design tokens from `@mfp-design-system/tokens`.

## Where to look

- **Tokens** — color, spacing, typography, and other primitives.
- **Components** — Lit-based custom elements built on top of tokens.

Stories are discovered from both this app's `stories/` folder and from
`packages/*/src/**/*.stories.ts` so each package can ship its own stories
alongside its source.
